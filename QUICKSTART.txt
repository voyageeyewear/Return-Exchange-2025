📦 INSTALLATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Install dependencies
  npm install
  cd frontend && npm install && cd ..

Step 2: Start backend (Terminal 1)
  npm start

Step 3: Start frontend (Terminal 2)
  cd frontend
  npm start

✅ Done! Open http://localhost:3000
